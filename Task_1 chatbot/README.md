# 🤖CampusBuddy - Student Helper Chatbot
CampusBuddy is a Python-based rule-based chatbot developed as part of the CodSoft Artificial Intelligence Internship.

It helps students with:
- Motivation
- Study tips
- Attendance calculation
- Time and date
- Programming guidance
- Fun jokes

## 🚀 Features
- Personalized Greeting
- Mood Detection
- Attendance Calculator
- Study Planner
- Motivation Quotes
- Time & Date Feature
- Python & DSA Guidance
- Interactive Terminal UI
- Typing Animation Effect

## 🛠 Technologies Used
- Python
- Colorama
- Datetime
- Random Module

## ▶ How to Run

```bash
python chatbot.py
```

## 📸 Sample Commands

- hello
- motivate me
- attendance
- joke
- study
- python
- time
- date
