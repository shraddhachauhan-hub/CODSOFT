# 🤖 ShraddhaXO AI - Tic-Tac-Toe Game

An AI-powered Tic-Tac-Toe game developed using **Python** and the **Minimax Algorithm** as part of the **CodSoft Artificial Intelligence Internship**.

## 🚀 Features
- Human vs AI Gameplay
- Unbeatable AI using Minimax Algorithm
- Interactive Terminal UI
- Live Scoreboard
- Replay Option
- Input Validation & Error Handling

## 🛠 Technologies Used
- Python
- Colorama
- Minimax Algorithm
- Game Theory

## ▶️ Run the Project

Install dependency:

```bash
pip install colorama
```

Run the game:

```bash
python tic_tac_toe_ai.py
```
