# VisionSpeak AI

VisionSpeak AI is an Image Captioning application built using Python, Tkinter, and Salesforce BLIP.

## Features
- Upload Image
- Generate AI Captions
- Normal & Detailed Caption Modes
- Text-to-Speech Output
- Save Captions
- User-Friendly GUI

## Technologies Used
- Python
- Tkinter
- Hugging Face Transformers
- BLIP Model
- PyTorch
- Pillow
- pyttsx3
