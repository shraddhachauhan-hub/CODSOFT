## 📂 Projects Completed

### 🤖 Task 1 – Student Helper Chatbot
A rule-based chatbot developed in Python that assists students by answering common queries and providing basic guidance through conversational interactions.

**Key Features:**
- Interactive text-based conversation
- Predefined responses for common student queries
- Simple and user-friendly interface

**Technologies Used:**
- Python
- Rule-Based Logic

### 🎮 Task 2 – ShraddhaXO AI (Tic-Tac-Toe AI)
An AI-powered Tic-Tac-Toe game that uses the Minimax Algorithm to make optimal decisions and provide a challenging gameplay experience.

**Key Features:**
- Human vs AI gameplay
- Intelligent move prediction
- Unbeatable AI using Minimax Algorithm

**Technologies Used:**
- Python
- Minimax Algorithm
- Colorama

### 🖼️ Task 3 – VisionSpeak AI
A computer vision project that detects objects in images and converts the detected information into speech, improving accessibility and user interaction.

**Key Features:**
- Image analysis
- Object detection
- Text-to-speech output

**Technologies Used:**
- Python
- OpenCV
- Text-to-Speech Libraries
- Computer Vision


### 🎬 Task 4 – Movie Recommendation System
A content-based recommendation system that suggests movies similar to the user's selected movie based on movie attributes and similarity calculations.

**Key Features:**
- Personalized movie recommendations
- Similarity-based filtering
- Fast and accurate suggestions

**Technologies Used:**
- Python
- Pandas
- Scikit-learn
- Recommendation Algorithms

### 😊 Task 5 – FaceGuard AI
A face detection and recognition system that identifies and recognizes faces using computer vision techniques.

**Key Features:**
- Real-time face detection
- Face recognition and identification
- Dataset creation and training support

**Technologies Used:**
- Python
- OpenCV
- NumPy
- Computer Vision
